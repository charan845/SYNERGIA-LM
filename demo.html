SYNERGIA-LM

Differentiable Neuro-Symbolic Knowledge Graph Integration with Constraint Propagation for Reliable Language Reasoning

Live Demo Architecture

7-Stage Pipeline

Click each stage to see details

01
Input Text / Question
Raw (subject, relation, ?) query from the KG completion task. The model receives a triple with missing object and must predict it.
02
Semantic Encoder
TF-IDF with character n-grams (2-4) for entity IDs and word-level features for relation paths. Produces 300-dim entity vectors and 500-dim relation vectors using scikit-learn.
03
Entity & Relation Extractor
Decomposes the query into subject entity and relation type. Identifies the domain of the relation (film, music, awards, geography, etc.) for targeted retrieval.
04
Knowledge Graph Retrieval
BFS-based k-hop subgraph extraction around the subject entity. Collects candidates from: (1) same-relation objects in training data, (2) similar-relation objects, (3) k-hop neighbors.
05 ⭐
Differentiable Symbolic Reasoning
Core Innovation: Fuzzy logic operations replace discrete logic gates — AND(a,b)=a·b, OR(a,b)=a+b−a·b, NOT(a)=1−a, IMPLIES(a,b)=1−a+a·b. Scores candidates via direct match, one-hop path reasoning, and type consistency — all differentiable.
06 ⭐
Constraint Propagation Engine
Core Innovation: Soft arc consistency — instead of hard domain pruning, uses continuous probability distributions. Applies functional constraints (one-to-one mapping), type constraints (known objects of same relation), and uniqueness constraints. Iterates until convergence.
07
Constraint-Aware Decoding
Merges reasoning scores with constraint-propagated probabilities using weighted geometric mean. Sorts candidates and returns top-k predictions with confidence scores.
OUT
Reliable Ranked Output

Technical Details

Implementation specifics

DIFFERENTIABLE LOGIC
AND a·b (product t-norm)
OR a+b−a·b (probabilistic sum)
NOT 1−a
IMPLIES 1−a+a·b
All continuous, gradient-friendly
CONSTRAINT PROPAGATION
Functional One-to-one mapping enforcement
Type Known object type boosting
Uniqueness Duplicate elimination
Iterates until convergence (<0.001 delta)
SEMANTIC ENCODER
Entity IDs: char n-grams (2,4), 300 features
Relations: word n-grams (1,2), 500 features
Sublinear TF-IDF weighting
Cosine similarity for matching
TOOLCHAIN
Python 3.14 | NLTK | NumPy | scikit-learn
9 modules, ~800 lines of code
Training: 272K triples, 2.16s build time
No GPU required, runs on CPU
